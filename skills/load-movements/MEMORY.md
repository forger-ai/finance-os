# Categorization Memory

Write short, durable mapping notes here when a merchant or pattern has already been classified and should be reused in future imports.

Examples:

- `Uber Eats` -> `No esencial / Delivery`
- `Lider` -> `Esencial variable / Supermercado`
- `Spotify` -> `Suscripciones / Streaming`
